# cURLとは
コマンドラインからさまざまなプロトコルを使ってデータを送受信するツール。  
主にAPIのテスト、ファイル転送、Webページの取得などに使われる、Client URLの略称。

## コマンド
```bash
curl <オプション> <URL>
-k # 証明書スキップ
-X <HTTPメソッド> # HTTPメソッドの指定
-H '<HTTPリクエストヘッダー>' # 任意のHTTPリクエストヘッダー
-d '<任意のデータ>' # 送信するデータの指定
-i # HTTPヘッダーの出力
-v # 詳細な通信情報の出力
-L # リダイレクトの追跡
-o # 出力をファイルに保存
-x # プロキシサーバーの指定
```
