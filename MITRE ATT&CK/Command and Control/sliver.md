# Sliver C2とは
攻撃者が被害者のシステムを遠隔操作し、悪意ある活動を行うために使うC2フレームワーク。

## ダウンロードから使用
```bash
wget https://github.com/BishopFox/sliver/releases/download/v1.5.44/sliver-server_linux
wget https://github.com/BishopFox/sliver/releases/download/v1.5.44/sliver-client_linux
sudo mv sliver-server_linux /usr/local/bin/sliver-server
sudo mv sliver-client_linux /usr/local/bin/sliver-client
sudo chmod +x /usr/local/bin/sliver-server
sudo chmod +x /usr/local/bin/sliver-client
```
